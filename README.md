# 🐍 Snake Game using Python & Tkinter

A simple **Snake Game** built using **Python** and **Tkinter**. This project is perfect for beginners who want to learn GUI programming, object-oriented programming (OOP), event handling, and game development basics in Python.

---

## 📌 Features

- 🐍 Classic Snake gameplay
- 🍎 Random food generation
- 🎨 Random food colors
- 📈 Live score tracking
- ⌨️ Keyboard controls
- 🔄 Restart button
- 💥 Collision detection (wall & self)
- 🖥️ Simple and beginner-friendly GUI

---

## 🛠️ Built With

- **Python 3**
- **Tkinter** (GUI Library)
- **Random Module**

No external libraries are required.

## 🧠 How the Game Works

1. A snake is created with three body parts.
2. Food appears at a random location on the screen.
3. Use the arrow keys to move the snake.
4. Eating food:
   - Increases your score.
   - Grows the snake.
   - Generates new food with a random color.
5. The game ends if:
   - The snake hits a wall.
   - The snake collides with its own body.
6. Press **Restart** to play again.
